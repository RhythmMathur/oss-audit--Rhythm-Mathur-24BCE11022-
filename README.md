## Description
This project contains an audit of Git and 5 shell scripts demonstrating Linux concepts.

## Scripts

### Script 1 — System Identity Report
Displays system information like kernel, user, uptime, etc.

### Script 2 — Package Inspector
Checks if Git is installed and shows the version.

### Script 3 — Disk Auditor
Shows permissions and size of important directories.

### Script 4 — Log Analyser
Counts occurrences of keywords in log files.

### Script 5 — Manifest Generator
Creates a personalised open-source philosophy statement.
